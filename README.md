# phonepicker
iPhone colour and size picker written with `AngularJS`.

## Requirements
* Node
* NPM

## Running the app
First run `npm install` to fetch the app dependancies.

Then the app uses Gulp, so to start the development server run..
```
gulp start
```
and to create production build
```
gulp build
```
